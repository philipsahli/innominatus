## Quick Start

### Prerequisites

- Go 1.21+
- Docker
- PostgreSQL (for testing)
- kubectl (for integration tests)

### Clone Repository

```bash
git clone https://github.com/philipsahli/innominatus.git
cd innominatus
```

### Build

See [Building from Source](building.md)

### Run Tests

```bash
go test ./...
```
