# Getting Started

```bash
# 1. Setup environment
./setup.sh

# 2. Start server
./innominatus

# 3. Open Web UI
open http://localhost:8081

# 4. Deploy your first app
./innominatus-ctl run deploy-app score-spec.yaml
```
